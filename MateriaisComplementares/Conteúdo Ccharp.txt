Aula 06: 25/10/2021
Código fonte base para realização da atividade prática: https://wordpad.cc/CSharp_Itau_Fonte_Base_Desafio01
Enunciado da atividade prática:
Construção de projeto para registrar vendas de produtos armazenados em uma tabela do banco de dados de nome produtos. A aplicação deverá ter as seguintes funções:
1) Cadastrar Novo Produto
2) Listar todos os produtos cadastrados
3) Listar produto de acordo com o código informado
4) Registrar venda, inserindo as seguintes informações:
	a) Nome do cliente
	b) ID do produto que será comprado
	c) quantidade do produto que será comprado

	Antes de registrar a venda, a aplicação deve verificar se existe estoque disponível do produto para realizar a venda. Se sim, a venda será registrada e o estoque debitado. Se não, uma mensagem de erro na tela será apresentada ao usuário.

Repositório GitHub Projeto Vendas: https://github.com/paponciano/projetovendas
